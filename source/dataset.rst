====================================================
Adding or editing sensors and observation locations
====================================================

Adding an observation location
============================================================
If the site has a more complex geographic setup you can describe additional locations and link them to a site record. This is done by creating or referencing to an "observation location".

It is mandatory to give a research site:

- Name of Observation Location
- Textual Description and
- Geographic Location (the actual boundaries of the area)

You can either create an observation location inline or go to the dedicated form ("Edit/Add new content" and then select "Observation Location")

Please don't create multiple observation locations for the same geographic boundaries.

Adding a sensor
============================================================
It is also possible to add sensors but currently that information is not further used and is purely informational
